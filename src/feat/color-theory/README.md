🎨 ColorAdvisor AI

A smart color recommendation platform built with Next.js that helps users select the most effective color palettes based on their website domain, target audience, and brand identity.

⸻

🚀 Overview

ColorAdvisor AI is a modern web application designed to simplify color selection for digital products. By analyzing user inputs such as industry domain, audience type, and brand tone, the application provides:
	•	Thoughtful color palette recommendations
	•	Clear explanations based on color psychology
	•	Practical UI usage guidelines
	•	Accessibility-focused suggestions

This tool is valuable for UI/UX designers, frontend developers, and clients who want to make better design decisions.

⸻

✨ Key Features
	•	🔍 Domain-based color recommendations (e.g., Real Estate, Finance, Healthcare, SaaS)
	•	🎯 Audience targeting (e.g., Young, Corporate, Premium)
	•	🎨 Automated palette generation (Primary, Secondary, Accent colors)
	•	🧠 Color psychology insights to justify design decisions
	•	💻 Live UI preview (buttons, cards, and layouts)
	•	🌗 Light and Dark mode support
	•	📋 Copy-ready HEX, RGB, and Tailwind CSS classes
	•	♿ Accessibility support with contrast checking
	•	⚡ Responsive and performant UI using modern frameworks

⸻

🧠 How It Works
	1.	The user provides:
	•	Website domain (e.g., Real Estate, E-commerce, SaaS)
	•	Target audience
	•	Brand tone (e.g., Modern, Luxury, Minimal)
	2.	The system processes these inputs and generates:
	•	A tailored color palette
	•	Contextual explanations based on design principles
	•	UI recommendations for effective implementation

⸻

💡 Example

Input:
	•	Domain: Real Estate
	•	Audience: Corporate
	•	Brand Tone: Luxury

Output:
